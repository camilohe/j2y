Simple CLI tool to convert JSON to YAML

```
PS > .\target\debug\j2y.exe --help
J 2 Y 0.0.1
Steven Murawski <steven.murawski@microsoft.com>
Converts JSON to YAML

USAGE:
    j2y.exe [FLAGS] <INPUT> <OUTPUT>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information
    -v               Include verbose output

ARGS:
    <INPUT>     Sets the input file to use (should be JSON)
    <OUTPUT>    Sets the output file to use (will be YAML)
```
